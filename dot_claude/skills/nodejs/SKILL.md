---
name: nodejs
description: Rules when developing javascript projects (in node or the browser)
---


* Use TypeScript
* Use `oclif` for cli
* Use `yarn` as package manager
* When installing a dependency, update the file `package.json.md` with why you added it.
* Use `zod` to create schema
* Use `zod` to create React props
* When creating schema with `zod`, add custom messages
